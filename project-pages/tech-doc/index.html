    Introduction

    This is a lightweight template that may be used for a technical documentation page. The available elements include: section heading, paragraph, code block, and bullet point. Each of these may be used as many times as necessary in a section.

    While building this site, the primary goals were high 'readibility' and simple navigation. For inspiration, I thought about different books I've read and their unique formats. For example, while in high school, I found reading textbooks unbearable--not because of the dry language, but because of all the colorful definitions, highlighted text-boxes, boldened examples, and sporadically placed charts/pictures. With so much going on in a single page, I couldn't keep focused on the text. This page is designed to be the opposite; with simple colors, consistent fonts, and a special dark mode, I think I hit the mark.

    Code blocks will appear like the example below:

    
  function intro(input) {
      let output = input.map(item => condition ? expressionA : expressionB);
      return output;
  };
